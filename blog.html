Learning Python the Fun Way

May 13, 2025

I've been diving deeper into Python lately—especially object-oriented programming. One thing that really helped me grasp classes and objects was thinking of them like real-world things. For example, I built a Car class with attributes like color and speed, and methods like drive() and brake(). It made the abstract ideas feel more concrete.

I’ve also been doing the 100 Days of Code Python Bootcamp by Angela Yu—and wow, it’s been a game changer! Each day introduces a new concept with a mini-project to apply it. From building a calculator and a coffee machine simulation, to working with APIs and GUIs—there's always something exciting to explore.

What I really love is how project-based the course is. I learn best by doing, and this bootcamp makes every concept feel practical and achievable.

Starting Flask Web Apps

May 15, 2025

One area I'm starting to really enjoy is web development with Python—and recently, I’ve begun exploring Flask. After doing a few hands-on class projects with Flask, I’ve become curious about taking things further on my own.

I’ve already built a couple of simple web apps in class—like a basic blog and a to-do list—and I’m now experimenting with layouts, forms, and routing. It’s exciting to see how everything connects behind the scenes, from back-end logic to front-end presentation.

I plan to keep building and sharing more Flask projects soon. I love how Python ties everything together!
